# API Gateway

RESTやWebSocket APIを作成して、AWSやAWSに保存しているデータや他のWEBサービスにアクセスすることができるサービス。

サーバレスインフラストラクチャの一部（フロントアプリ側）になる。認証も可能。

## 料金
