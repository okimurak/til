# tfenv

[tfutils/tfenv: Terraform version manager](https://github.com/tfutils/tfenv)

Terraform のバージョンを管理する

## Install

Homebrew では

```bash
brew unlink terraform
brew install tfenv
```

## How to use

インストールとバージョン設定

```bash
tfenv install <Terraform Version>
tfenv use <Terraform Version>
```

一覧表示

```bash
tfenv list
```
