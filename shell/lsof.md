# lsof

オープンしているファイルを一覧表示する

## 現在開いているファイルを表示

```
lsof <パス>
```

## ユーザを指定する

`-u`オプションを使う。続けて書けば、複数指定でもできる

```
lsof -u study -u pikachu
```

## プロセスを指定する

`-p`(プロセスID), `-c`(プロセス名)オプションを用いる

```
lsof -p
```

## ポート、IPを指定する

`-i` オプションを使う

```
lsof -i

lsof -i:<Port Num> # 指定したポートを利用しているプロセス
```
